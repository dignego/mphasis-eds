## Migration Approach

The migration will use the **site migration** workflow which:
1. Analyzes site structure and identifies page templates (groups of similarly-structured pages)
2. Analyzes individual pages to identify blocks and content patterns
3. Generates import infrastructure (parsers and transformers)
4. Executes bulk content import to produce EDS-compatible HTML
5. Migrates the visual design (CSS/styling)
